# Scrcpy GUI (WinUI 3 版)

这是一个使用 C# 和 WinUI 3 开发的现代 scrcpy 图形化控制面板。

## 🌟 项目特色

- 🎨 **WinUI 3 风格界面**：采用 Windows 11 Fluent Design 原生 Mica 亚克力背景与卡片化布局。
- ⚡ **自动下载托管**：支持自动下载并配置最新版 `scrcpy` 运行组件，亦可手动指定本地路径。
- 📱 **设备与无线连接**：支持 USB 和无线 ADB 连接管理，包含 Android 11+ 无线调试配对助手。
- 🖥️ **参数配置与日志**：提供直观的比特率、分辨率和帧率设置，并内嵌实时控制台日志窗口。

## 🚀 快速开始

1. 前往 [Releases 页面](https://github.com/tom613951/scrcpy-gui-winui3/releases) 下载最新的 `scrcpy-gui-winui3-v1.0.0-x64-Portable.zip`。
2. 解包到本地目录，双击运行 `ScrcpyGui.exe`。
